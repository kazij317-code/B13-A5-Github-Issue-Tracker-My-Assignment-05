1️⃣ What is the difference between var, let, and const?

Ans: Difference betewen var, let and const:
var: function scoped, can be re-declared and updated, hoisted with undefined.

let: Block scoped, can be updated but not re-declared in the same scope.

const: block scoped, Can't be updated or redeclared, value must be assigned when declared


2️⃣ What is the spread operator (...)?

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans:
map(): Transform each element, returns new array

filter(): select elements based on condition, returns new array

forEach(): looping through elements and returns nothing
